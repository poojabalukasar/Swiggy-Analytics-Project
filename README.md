# Swiggy-Analytics-Project
End-to-end analytics project using Excel, SQL, Power BI &amp; Tableau.


This project focuses on analyzing Swiggy food delivery data to extract meaningful business insights and support data‑driven decision‑making. 
The analysis covers order trends, customer behavior, restaurant performance, and revenue metrics. 
Dashboards were created using Excel, Power BI, and Tableau, and insights were validated using SQL queries.


# Business Objective
1. Analyze total orders and revenue performance
2. Identify top-performing cities and restaurants
3. Understand customer ordering behavior
4. Track order and revenue trends over time
5. Calculate key business KPIs for decision-making


# Tools & Technologies
1. # Excel: Data cleaning, pivot tables, KPI calculations, and dashboard creation
2. # SQL: Data querying, joins, aggregations
3. # Power BI: Interactive dashboards, DAX measures, data modeling
4. # Tableau: Visual analytics, filters, and data storytelling


# Key KPIs
Total Orders 
Total Revenue 
Total Customers 
Top Performing Restaurants
Order Cancellation Rate
Active Delivery Agents 
Top Food Categories 
Peak Order Hours
Average Delivery Time


# Dashboards 
1.  Excel Dashboard
2.  Power BI Dashboard
3.  Tableau Dashboard


# Key Insights
1. Orders peak during weekends and evening hours
2. A few top cities contribute a major share of revenue
3. High-performing restaurants generate repeat orders
4. Returning customers have higher average order value
